## Sobre esta Certificação Profissional

A ciência de dados é uma das profissões mais promissoras da década, e a demanda por cientistas de dados que podem analisar dados e comunicar resultados para informar decisões baseadas em dados nunca foi tão grande. Este Certificado Profissional da IBM ajudará qualquer pessoa interessada em seguir carreira em ciência de dados ou aprendizado de máquina a desenvolver habilidades e experiência relevantes para a carreira.

O programa consiste em 9 cursos on-line que fornecerão as mais recentes ferramentas e habilidades prontas para o trabalho, incluindo ferramentas e bibliotecas de código aberto, Python, bancos de dados, SQL, visualização de dados, análise de dados, análise estatística, modelagem preditiva e algoritmos de aprendizado de máquina. Você aprenderá ciência de dados por meio da prática no IBM Cloud usando ferramentas de ciência de dados reais e conjuntos de dados do mundo real.

Este Certificado Profissional tem uma forte ênfase na aprendizagem aplicada. Com exceção do primeiro curso, todos os outros cursos incluem uma série de laboratórios práticos no IBM Cloud que fornecerão habilidades práticas com aplicabilidade a trabalhos reais, incluindo: 

**Tools**: Jupyter / JupyterLab, GitHub, R Studio, and Watson Studio 

**Libraries**: Pandas, NumPy, Matplotlib, Seaborn, Folium, ipython-sql, Scikit-learn, ScipPy, etc. 

**Projects**: random album generator, predict housing prices, best classifier model, Predicting successful rocket landing, dashboa rd and interactive map
